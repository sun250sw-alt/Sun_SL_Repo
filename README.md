# Sun SL Repository

Kodi 21 (Omega) addon repository by [sun250sw-alt](https://github.com/sun250sw-alt).

## Addons

| Addon | Version | Description |
|---|---|---|
| **Subtitle Translator** | 1.1.0 | Translate subtitles via Gemini AI or Google Translate from the Download Subtitles menu |

---

## How to install

### Step 1 — Download the repository addon

Download this file directly to your device:

**[⬇ Download repository.sun_sl_repo-1.0.0.zip](https://raw.githubusercontent.com/sun250sw-alt/Sun_SL_Repo/main/repository.sun_sl_repo/repository.sun_sl_repo-1.0.0.zip)**

### Step 2 — Enable Unknown Sources in Kodi
Settings → System → Add-ons → Unknown Sources → ON

### Step 3 — Install repository from the downloaded ZIP
Settings → Add-ons → Install from ZIP file → browse to the downloaded file → select it

Wait for **Sun SL Repository installed** notification.

### Step 4 — Install Subtitle Translator from the repository
Settings → Add-ons → Install from repository → **Sun SL Repository** → Subtitles → **Subtitle Translator** → Install

## Gemini AI Setup (Optional)

1. Go to **aistudio.google.com** → sign in → **Get API Key** → **Create API Key** → copy it
2. In Kodi: play a video → press T → Download Subtitles → **Subtitle Translator**
3. Tap **Gemini API Key [Not set]** → paste key → confirm
4. Tap **Gemini Model** to choose model

### Free tier limits (resets daily midnight Pacific Time):
| Model | Requests/day | ~Movies/day |
|---|---|---|
| Gemini 2.5 Flash *(recommended)* | 250 | ~10 |
| Gemini 2.5 Flash-Lite | 1000 | ~43 |
| Gemini 2.5 Pro | 100 | ~4 |

If limit reached → automatically falls back to Google Translate.
