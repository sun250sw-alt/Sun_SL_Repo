# Sun SL Repository

Kodi 21 (Omega) addon repository.

## Addons

| Addon | Description |
|---|---|
| **Subtitle Translator** | Translate subtitles via Gemini AI or Google Translate from the Download Subtitles menu |

## Install in Kodi

### Step 1 — Add source
Settings → File Manager → Add source
```
https://raw.githubusercontent.com/sun250sw-alt/Sun_SL_Repo/main
```
Name it: `Sun SL Repo`

### Step 2 — Install repository
Settings → Add-ons → Install from ZIP → Sun SL Repo →
`repository.sun_sl_repo` → `repository.sun_sl_repo-1.0.0.zip`

### Step 3 — Install addon
Settings → Add-ons → Install from repository → Sun SL Repository →
Subtitles → Subtitle Translator → Install

## Usage
Play a video → press T → Download Subtitles → Subtitle Translator
